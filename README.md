# ALG---2026.01---3EM1
Atividade proposta pelo professor Rafael, com o objetivo de ensinar o uso de funções customizadas em C.

##Exigência
O aluno deve desenvolver a lista de exercício em linguagem C utilizando o DEV-C++.

    Crie um repositório PUBLICO no github
    Crie as funções em um único código local SEM utilizar nenhuma variável global.
        Calcular H:m;s
        isPar (verificar se o número é par, retorna true se é par e false se é impar)
        isImpar ( verificar se o número é ímpar, retorna true se é ímpar, e false se é par)
        media (calcula média entre dois números)
        maior (retorna o maior valor de dois números)
        menor (retorna o menor valor de dois números)
    Publique os códigos no seu repositório GitHub
    Preencha o questionário com o link do seu repositório.
